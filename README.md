MyTask adalah aplikasi yang digunakan untuk mengelola atau management tugas harian

1. Login Page
   Pengguna dapat masuk ke dalam sistem dengan memasukkan username dan password yang valid.
2. Dashboard
   Menampilkan halaman utama setelah login dengan:
   - Navbar navigasi ke halaman Dashboard, Pengelolaan, dan Profil.
   - Sedikit teks terkait mytask Tombol Get Started yang ketika di klik mengarah ke pengelolaan, dan leanrn more
3. Halaman Pengelolaan 
   Berisi daftar tugas yang ditampilkan dalam bentuk list dengan status masing-masing (misal: selesai, sedang dikerjakan, belum dikerjakan).
4. Profile 
   Menampilkan informasi data diri dari pengguna yang sedang login.
